# Hepburn Romaji Standards

## System
Modified Hepburn (Revised) with macrons for long vowels.

## Long Vowel Rules
| Sound | Romaji | NOT this |
|-------|--------|----------|
| おう/おお | ō | ou, oo |
| うう | ū | uu |
| いい | ii | ī (ii is correct in Hepburn) |
| ええ | ē | ee |
| ああ | ā | aa |

## Examples
- 東京 → Tōkyō (not Toukyou, not Tokyo)
- 大阪 → Ōsaka (not Oosaka)
- 経済 → keizai (no long vowel)
- 業績 → gyōseki (not gyouseki)
- 低調 → teichō (not teichou)
- ニュース → nyūsu (not niyuusu, not niyusu)
- 状況 → jōkyō (not joukyou)

## Particle Rules
- は (topic) → wa (not ha)
- へ (direction) → e (not he)
- を (object) → wo (not o)

## Capitalization
- Proper nouns capitalized: Doitsu, Nihon, Amerika, Merz, Tōkyō
- Sentence-initial capitalization
- Everything else lowercase

## Special Cases
- ん before b/m/p → m: shinbun → shimbun, sampō
- ん before vowel or y → n': shin'ichi, ken'ō
- っ (double consonant) → double the consonant: kitte, happyō, zasshi

## Common Corrections the Tutor Should Make
| User writes | Correct form | Note |
|------------|-------------|------|
| niyusu | nyūsu | ny + macron |
| toukyou | Tōkyō | macron, capitalize |
| dochia | Doitsu | completely wrong romaji |
| joukyou | jōkyō | macron |
| teichou | teichō | macron |
| konshuu | konshū | macron |
