# Kana Writing Standards

## System
Hiragana + katakana only. No kanji, no romaji.

## Word Spacing
Add a space between each word/particle for readability:
- しゅしょう は あたらしい けいざい せいさく を はっぴょう しました。
- NOT: しゅしょうはあたらしいけいざいせいさくをはっぴょうしました。

## Katakana Usage
Use katakana for:
- Loanwords: コンピューター、ロボット、テクノロジー
- Foreign names: ドイツ、アメリカ、メルツ
- Company names: ソフトバンク、トヨタ
- Onomatopoeia (optional)

## Hiragana Usage
Use hiragana for:
- Native Japanese words: けいざい、せいじ、しゃかい
- Particles: は、が、を、に、で、と、も、か
- Verb conjugations: します、できます、なりました
- Adjectives: あたらしい、おおきい

## Long Vowel Rules
- おう sounds: おう (not oo): とうきょう、けいざい
- うう sounds: ゆう、くう
- Katakana long vowels use ー: ニュース、コンピューター

## Capitalization
- Katakana naturally stands out, no capitalization needed
- Sentence-initial words stay lowercase in hiragana

## [NEW] and [REVIEW] Markers
Place markers before the word, same as romaji mode:
- [NEW]しゅしょう は [REVIEW]けいざい の [NEW]せいさく を はっぴょう しました。

## Particle Spelling
- Topic は stays as は (not わ)
- Direction へ stays as へ (not え)
- Object を stays as を (not お)
